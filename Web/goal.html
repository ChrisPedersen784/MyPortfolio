Web Development

Learning Goals

Kortsigtede læringsmål

Webudvikling med React:

  • Forstå grundlæggende React-koncepter som komponenter, props, state og livscyklusmetoder.
  • Kunne oprette en simpel React-applikation med JSX og komponentstruktur.
  • Forstå og anvende tilstandsstyring med React Hooks som useState og useEffect.
  • Integrere eksterne API'er i React-apps og vise dynamiske data.
  • Implementere formularer og håndtere brugerinput ved hjælp af kontrolleret komponenttilstand.
  • Styling af React-komponenter ved hjælp af CSS eller et CSS-in-JS bibliotek som styled-components.

Langsigtede læringsmål

Webudvikling med React:

  • Bygge avancerede React-applikationer med komplekse brugergrænseflader.
  • Forstå og implementere asynkrone operationer som API-kald og promises.
  • Integrere autentificering og autorisation i React-applikationer.
  • Implementere testning ved hjælp af enheds- og integrationstestværktøjer og React Testing Library.
